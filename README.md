# CocoRaytrace

Porting of a simple raytracer on a Tandy Coco 3

<img src="CocoRT.png">

# How to build
- Install [CMOC](http://perso.b2b2c.ca/~sarrazip/dev/cmoc.html)
- Adapt [COCOMakefile](./COCOMakefile)
- Make -f COCOMakefile run

# Now you are on the COCO side (XRoar)
- LOADM "BGRAY"
- EXEC
- WAIT.................
